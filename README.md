# UE-OpenMAIC

This project is based on **OpenMAIC**.

UE-OpenMAIC repository: https://github.com/zt282/UE-OpenMAIC

OpenMAIC repository: https://github.com/THU-MAIC/OpenMAIC

## Repository Structure

```text
UE-OpenMAIC/
+-- baseline/
|   +-- .gitkeep
+-- dataset/
|   +-- .gitkeep
+-- benchmark/
|   +-- .gitkeep
+-- code/
|   +-- .gitkeep
+-- OpenMAIC/
    +-- OpenMAIC application source code
```

## Directory Notes

- `OpenMAIC/`: the main OpenMAIC application code.
- `code/`: additional code artifacts and extensions.
- `baseline/`: baseline methods and reproduction materials.
- `dataset/`: datasets or dataset preparation notes.
- `benchmark/`: benchmark definitions, scripts, and evaluation outputs.
